RFID identification system

Identification system for the Autonomous University of Queretaro

IoT
MQTT
Embeeded systems

This system lets the students of the Faculty of Computer Sciences of the Autonomous University of Querétaro easily use RFID tags as identification for different administrative procedures.

Using IoT technologies, this system was the result of a collaboration between a group of students from different majors of the Faculty. I was responsible of the back-end server listening and handling all the MQTT requests that the RFID scanning boxes send.

I also programmed the configuration system, which assigns each RFID scanning box its function. With this system, users can easily change the configuration of each box remotely and use this identification method for any other system needed.

It was built mainly for The Academic and Cultural Week. This is an important event that takes place every year in the Faculty of Computer Sciences. It is a week where students attend special workshops and lectures given by guest professors instead od regular classes. At the end of these workshops and lectures, each student is given a certain amount of points depending o the importance of the event.

This way, at the end of this week, students with the most amount of points gathered are given special prices, like speakers, headphones, hard drives... even drones.

For more information about this event and the development of the UX of the front-end application, visit UX Case Study by Fernando OC

The problem was that points were given through codes printed on paper, which could be redeemed through the Faculty web page. This allowed groups of students to organize themselves to cheat by giving all their codes to just one student, so that person would get a really high amount of points, winning the best price. Also, some people would just show up to end of the events and ask for a code.

With the system we built, now each student is assigned a RFID tag. This way, they now need to use their RFID tags to check-in and check-out of every event. The RFID scanning boxes connected to the campus WIFI then send a request to the server and adds the points automatically to the student if all the validations pass.

It is planned to implement this authentication system in many more administrative procedures of the Faculty.
