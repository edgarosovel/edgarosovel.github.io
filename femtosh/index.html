Femtosh

Small shell written in bash

Made with Bash

Femtosh is a small shell (femto = 10e−15), written in Bash. Femtosh lets you do all this things:

    🔹di    ➩ Say a text (text to speech). macOS only

    🔹l    ➩ List files and folders from current directory

    🔹cl    ➩ Clear screen

    🔹copiar    ➩ Copies given files to the given route

    🔹ir    ➩ Go to given directory

    🔹renombrar    ➩ Renames a file or directory

    🔹permisos    ➩ Lists and modifies files' permissions

    🔹exe    ➩ Creates a new exexcutable file

    🔹crear    ➩ Creates a new file or directory

    🔹eliminar    ➩ Deletes given files or directories

    🔹ver    ➩ Visualize a file or the content of a directory

    🔹editar    ➩ Edit a file on VI

    🔹procesos    ➩ Lists current processes

    🔹cerrar    ➩ Closes given process

    🔹ayuda    ➩ Lists all commands

    🔹salir    ➩ Closes the shell 𐊇𐌴𐊎𐊗𐊒·∫h
